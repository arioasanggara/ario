# Porto Rio - Personal Portfolio

Portofolio profesional milik **Ario Anggara**, seorang Food Consultant & Entrepreneur. Website ini dirancang dengan estetika modern, mewah (Gold & Dark theme), dan performa tinggi menggunakan teknologi web native.

## 🚀 Fitur Utama

- **Modern UI/UX**: Desain premium dengan efek *Glassmorphism* dan palet warna emas CR7.
- **Responsive Design**: Optimal diakses dari perangkat mobile, tablet, maupun desktop.
- **Scroll Reveal Animations**: Animasi halus saat scroll menggunakan *Intersection Observer API*.
- **Mobile Optimized Navigation**: Menu navigasi yang intuitif untuk pengguna smartphone.
- **Clean Code**: Struktur folder yang rapi dan kodingan JavaScript yang modular.

## 🛠️ Tech Stack

- **HTML5**: Struktur semantik untuk SEO yang lebih baik.
- **Tailwind CSS**: Framework CSS utility-first untuk styling yang cepat dan konsisten.
- **Vanilla JavaScript**: Logika interaksi tanpa ketergantungan library eksternal (ringan).
- **Google Fonts**: Menggunakan font 'Montserrat' untuk kesan elegan.

## 📂 Struktur Folder

```text
/porto-rio/
├── index.html              # Halaman utama
├── assets/                 # Aset statis proyek
│   ├── js/                 # Logika JavaScript
│   │   └── script.js
│   └── img/                # Gambar dan Icon
└── README.md               # Dokumentasi proyek
```

## ⚙️ Persiapan Lokal

Untuk menjalankan proyek ini di komputer Anda:

1. Clone atau download repository ini.
2. Pastikan struktur folder sesuai dengan bagan di atas.
3. Buka file `index.html` menggunakan browser pilihan Anda (Chrome, Edge, Firefox).
4. *Disarankan*: Gunakan extension **Live Server** di VS Code agar perubahan kodingan langsung terlihat secara real-time.

## 📝 Catatan Pengembangan

- **Tailwind CDN**: Proyek ini menggunakan CDN untuk kemudahan pengembangan cepat. Untuk produksi skala besar, disarankan menggunakan Tailwind CLI.
- **Aset Gambar**: Gambar saat ini menggunakan placeholder dari Unsplash. Pastikan untuk menggantinya dengan foto asli di folder `assets/img/`.
